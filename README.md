# AQI-Predictor
An Air Quality Index prediction and visualization project that analyzes CSV datasets, predicts air quality using Machine Learning, and generates interactive charts for data exploration.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Matplotlib
- Streamlit

 # Features
- Import AQI datasets from CSV files
- Data preprocessing and analysis
- AQI prediction using Machine Learning
- Interactive visualizations with Plotly
- Statistical graphs using Matplotlib
- Easy-to-use workflow for exploring air quality data
